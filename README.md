# cardwire-noctalia

> [!IMPORTANT]
> This plugin is under acticve development, as is the plugin api for Noctalia v5. If you see this messgage, expect things to break!

Minimum noctalia version: `5.0.0`.

Brings GPU control to your noctalia shell.

Made possible by [cardwire](https://github.com/OpenGamingCollective/cardwire).  
Check out [noctalia](https://github.com/noctalia-dev/noctalia) for a great shell.

## Quick development setup

Follow [plugin development overview](https://docs.noctalia.dev/v5/plugins/development/).
